are any appointments within 15 minutes or not.

MAIN MENU:
After the appointment notification, the user will be directed to the main menu where 5 buttons will appear on the screen.

APPOINTMENTS:
The appointment button will lead the user to a screen that displays all existing appointments by default. The user can select from all appointments, the current week and the
current month appointments by selecting the appropriate radio button. Underneath the table on the left-hand side, there are 3 buttons. The add button allows you to add
new appointments, the update button will allow users to update the selected appointment and the delete button will allow users to delete the selected appointment from the
database. The back button will take you back to the main menu screen.

CUSTOMERS:
The customer button will take the user to the customer list and provide a table with all the available customer's information. Underneath the table on the left-hand side,
there are 3 buttons. The add button will allow you to add a new customer, the update button will allow you to update the selected customer, and delete will allow you to
remove the selected customer from the system. The back button will take you back to the main menu screen.

REPORTS:
The report button will take you to the report list where there will be three tabs. Each tab has its own report,Type/Month Report, Contact Schedule Report and Customers by
 Location Report. The Type/Month Report tab displays the total appointments by appointment type on the left side. On the right side, the table displays total appointments by
 month. The Contact Schedule Report tab has a combobox that allows you to select a contact and will display appointments associated with the selected contact in the table.

REPORT A3F: The Customer Totals By Location tab shows the number of customers per location.

The back to menu button will take you back to the main menu screen.

MySQL Connector Driver Version: mysql-connector-java-8.0.28
